# Penguen Store

Mern Stack based e-commerce site
  - Mysql
  - Expressjs
  - React
  - Nodejs

# Installation
inside server folder
  ``` 
    npm install
  ```
  and inside client folder
  ``` 
    npm install
  ```
 # Run
 ### Server
 server runs on localhost 4000 make sure to configure DB connection configs
 ```
 nodemon index.js
 ```
### Client
 ```
 npm start
 ```
